# DaBomb - API

This is a RESTful API for connecting the DaBomb [front end](https://github.com/cbachich/dabomb-fe) to a PostgreSQL database.

## API Endpoints

Index
```
GET /annotations
```

Show
```
GET /annotations/:id
```

Create
```
POST /annotations
```

Update
```
PATCH /annotations/:id
```

Destroy
```
Destroy /annotations/:id
```
